Bakeology

Bakeology is a cake-baking website build using django.
This takes order and passes the order via email using Formspree.
(Formspree is a thirdparty mail sender)
This also allows clients to post comments that will be vrified
by admin and will be added after verification.
In progress adding payment gateway that allows user
